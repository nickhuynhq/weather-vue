## Installation

1. Clone this repository.
```$ git clone https://git@github.com:nickhuynhq/weather-vue.git```


2. Run `npm install` from inside the client directory.
```bash
$ cd weather-vue
$ npm install
```

3. Create your own .env file
```
  VITE_API_KEY=<YOUR_API_KEY>
```

4. Compile Vue App

```bash
$ npm run dev
```
